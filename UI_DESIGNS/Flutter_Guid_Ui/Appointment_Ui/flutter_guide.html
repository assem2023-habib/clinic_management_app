🎬 قائمة جميع التأثيرات في الصفحة
3 AnimationControllers FadeTransition SlideTransition ScaleTransition AnimatedContainer
⏱ الخط الزمني للأنيميشن عند فتح الصفحة

0ms initState

يبدأ _headerFadeController.forward() و _slotListController.forward()

0→600ms FadeTransition — الهيدر

AppBar يظهر تدريجياً من opacity=0 إلى opacity=1 مع Curves.easeOut

0→400ms Slide + Fade — قائمة المواعيد

القائمة تنزلق من أسفل (Offset 0,0.08) مع Fade في نفس الوقت

عند الضغط Scale — زر التأكيد

يُضغط الزر لـ 0.95 عند onTapDown، ثم يعود عند onTapUp (150ms)

عند اختيار يوم إعادة تشغيل — قائمة المواعيد

_slotListController.forward(from: 0) لإعادة تشغيل أنيميشن الـ Slide

🔧 كيفية إنشاء الـ Controllers
// 1. أعلن عن الـ controllers في الـ State late AnimationController _headerFadeController; late AnimationController _confirmBtnController; late AnimationController _slotListController; // 2. أعلن عن الـ Animations late Animation<double> _headerFade; late Animation<double> _confirmScale; late Animation<double> _slotFade; late Animation<Offset> _slotSlide;
// 3. هيّأها في _initAnimations() void _initAnimations() { // هيدر الفيد _headerFadeController = AnimationController( vsync: this, duration: const Duration(milliseconds: 600), )..forward(); // ابدأ فور الإنشاء _headerFade = CurvedAnimation( parent: _headerFadeController, curve: Curves.easeOut, ); // زر التأكيد (Scale) _confirmBtnController = AnimationController( vsync: this, duration: const Duration(milliseconds: 150), ); _confirmScale = Tween<double>(begin: 1.0, end: 0.95) .animate(_confirmBtnController); // قائمة المواعيد (Slide + Fade) _slotListController = AnimationController( vsync: this, duration: const Duration(milliseconds: 400), )..forward(); _slotFade = CurvedAnimation( parent: _slotListController, curve: Curves.easeOut, ); _slotSlide = Tween<Offset>( begin: const Offset(0, 0.08), end: Offset.zero, ).animate(CurvedAnimation( parent: _slotListController, curve: Curves.easeOut, )); }
// 4. لا تنسَ dispose() — لمنع memory leaks @override void dispose() { _headerFadeController.dispose(); _confirmBtnController.dispose(); _slotListController.dispose(); super.dispose(); }

🔗 تهيئة الاتصال بـ Firebase
// في initState void _updateFirebaseRef() { // بناء مفتاح التاريخ: "2026-06-13" final dateKey = '${_selectedDate.year}-' '${_selectedDate.month.toString().padLeft(2, "0")}-' '${_selectedDate.day.toString().padLeft(2, "0")}'; // مرجع Node المواعيد _slotsRef = FirebaseDatabase.instance .ref('appointments/${widget.doctorId}/$dateKey/slots'); // Stream يستمع للتغييرات المباشرة _slotsStream = _slotsRef.onValue; }
📡 الاستماع للبيانات في الـ build
// في دالة build() استخدم StreamBuilder StreamBuilder<DatabaseEvent>( stream: _slotsStream, builder: (context, snapshot) { if (snapshot.hasData) { final data = snapshot.data!.snapshot.value as Map<dynamic, dynamic>?; // تحديث الـ State بعد الـ build WidgetsBinding.instance.addPostFrameCallback( (_) => _parseSlots(data) ); } return _buildBody(); // يُعيد رسم الصفحة }, ),
✍️ تحديث موعد في Firebase
// عند اختيار Slot جديد Future<void> _selectSlot(AppointmentSlot slot) async { // ① Optimistic UI — تحديث فوري محلياً setState(() { for (var s in [..._morningSlots, ..._eveningSlots]) { if (s.status == SlotStatus.selected) s.status = SlotStatus.available; // إلغاء القديم } slot.status = SlotStatus.selected; _selectedSlotTime = slot.time; }); // ② إرسال التحديث لـ Firebase try { final Map<String, dynamic> updates = {}; for (var s in [..._morningSlots, ..._eveningSlots]) { updates[_arabicToKey(s.time)] = s.toMap(); } await _slotsRef.update(updates); } catch (e) { debugPrint('خطأ Firebase: $e'); } }
✅ تأكيد الحجز النهائي
// تحديث الـ slot من "selected" إلى "booked" await FirebaseDatabase.instance .ref('appointments/${widget.doctorId}/$dateKey/slots/$slotKey') .set({'status': 'booked'});
💡 مهم: بعد تأكيد الحجز، سيتلقى جميع المستخدمين الآخرين الذين يشاهدون نفس الطبيب ونفس اليوم التحديث تلقائياً عبر onValue stream!

🃏 _GlassCard — التأثير الزجاجي

في Flutter لا يوجد backdropFilter بسهولة الـ CSS، لكن نحاكيه بالألوان الشفافة:

class _GlassCard extends StatelessWidget { final Widget child; @override Widget build(BuildContext context) { return ClipRRect( borderRadius: BorderRadius.circular(16), child: BackdropFilter( filter: ImageFilter.blur(sigmaX: 12, sigmaY: 12), child: Container( padding: const EdgeInsets.all(20), decoration: BoxDecoration( // rgba(3,37,21,0.4) — Glass Effect color: const Color(0x66032515), borderRadius: BorderRadius.circular(16), border: Border.all( color: Colors.white.withOpacity(0.08), ), ), child: child, ), ), ); } }
⚠️ BackdropFilter يحتاج import 'dart:ui' في أعلى الملف.

✅ ملاحظات مهمة
• الـ Slot أوقاتها مخزنة بمفاتيح إنجليزية في Firebase (09:00) وتُعرض بالعربية (٠٩:٠٠ ص)
• استخدم TickerProviderStateMixin (وليس SingleTicker) لأن لدينا أكثر من AnimationController
• Optimistic UI يُحسّن تجربة المستخدم — التحديث يظهر فوراً قبل انتهاء Firebase
• الـ StreamBuilder يُعيد بناء الواجهة تلقائياً عند أي تغيير في قاعدة البيانات في الوقت الحقيقي
